## ioBroker Adapter

Since you chose ioBroker as your data source, you need to install the SOLECTRUS ioBroker Adapter. It writes the measurements directly to the InfluxDB of SOLECTRUS.

**Link to the adapter:**

https://github.com/patricknitsch/ioBroker.solectrus-influxdb

**Installation:**

1. Install the adapter via the ioBroker Admin
2. Configure the connection to the SOLECTRUS InfluxDB
3. Select the data points to be transmitted
