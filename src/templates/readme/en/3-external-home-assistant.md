## Home Assistant Integration

Since you chose Home Assistant as your data source, you need to install the SOLECTRUS Home Assistant Integration. It writes the measurements directly to the InfluxDB of SOLECTRUS.

**Link to the integration:**

https://github.com/solectrus/ha-integration

**Installation:**

1. Install the integration via HACS or manually
2. Configure the connection to the SOLECTRUS InfluxDB
3. Select the sensors to be transmitted
